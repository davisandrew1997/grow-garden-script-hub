## What It Does

Grow a Garden Hub is a script utility made for the Grow a Garden experience in an HTML-oriented environment. It is organized as a hub so automation helpers stay in one place, giving users a cleaner way to manage script-based actions that can cut down on repetitive in-game work.

The release focuses on two main action sets: aimbot support and auto-collect behavior. Rather than acting like a fixed macro, the structure is intended to stay configurable, so the utility can be tuned for different play styles and updated as the surrounding environment changes.

- Aimbot support for target-oriented gameplay assistance
